# ATM Management System

A console-based ATM Management System developed in Java.

## Features

- Login with PIN
- Deposit Money
- Withdraw Money
- Check Balance
- Change PIN
- Three Login Attempts
- Input Validation
- Object-Oriented Design
- Inheritance

## Technologies Used

- Java
- OOP
- Inheritance
- Scanner Class

## Future Improvements

- Transaction History
- File Handling
- Database Integration
- GUI Version
