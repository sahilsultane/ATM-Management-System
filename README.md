## Features

- 🔐 Secure PIN authentication
- 💰 Deposit money
- 💸 Withdraw money
- 💳 Balance inquiry
- 🔄 Change PIN
- 🚫 Three login attempts
- 🎨 Colored terminal output
- 🏛 Built using Inheritance & Encapsulation

```bash

cd ATM-Management-System

javac *.java

java ATMManagementSystem
```
